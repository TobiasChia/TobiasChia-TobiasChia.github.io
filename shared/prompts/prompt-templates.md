# AI故事情节工具 - 提示词模板库

## 角色创建提示词模板

### 基础角色生成模板
```
你是一个专业的故事创作助手。请根据以下信息创建一个详细的角色：

角色类型：{characterType}
故事类型：{storyGenre}
背景描述：{characterBackground}

请为这个角色生成以下内容：
1. 基本信息（姓名、年龄、外貌特征）
2. 性格特点（至少3个主要特质，包括优点和缺点）
3. 背景故事（成长经历、重要事件、转折点）
4. 动机和目标（角色的内在驱动力和外在目标）
5. 弱点和恐惧（角色的脆弱面和内心恐惧）
6. 特殊技能或能力（专业技能、天赋、超能力等）
7. 人际关系（家庭、朋友、敌人、导师等）
8. 角色弧线（角色在故事中的成长轨迹）

请确保角色有血有肉，富有立体感和可信度。避免刻板印象，创造独特而有趣的角色。
```

### 高级角色生成模板（包含心理分析）
```
作为一名资深角色心理学家和故事创作专家，请基于以下参数创造一个复杂而真实的角色：

## 角色基础信息
- 角色类型：{characterType}
- 故事类型：{storyGenre}
- 时代背景：{timeSettingOr}
- 文化背景：{culturalBackground}
- 社会地位：{socialStatus}

## 详细背景
{characterBackground}

## 创作要求
请创造一个包含以下层次的角色：

### 1. 表层身份（他人眼中的形象）
- 外貌描述（身高、体型、特征、着装风格）
- 职业和社会角色
- 公开的性格特征
- 社交表现

### 2. 内在核心（真实自我）
- 核心价值观和信念
- 深层恐惧和不安
- 隐藏的欲望和野心
- 童年创伤或关键经历

### 3. 心理机制
- 防御机制（如何保护自己）
- 应对压力的方式
- 决策模式
- 情感表达方式

### 4. 关系动态
- 与不同类型人的互动模式
- 信任和背叛的界限
- 爱的表达和接受方式
- 权力关系中的行为

### 5. 角色发展轨迹
- 故事开始时的状态
- 面临的核心冲突
- 必须克服的内在障碍
- 可能的成长方向

请确保角色的所有方面都相互呼应，形成一个有机的整体。每个细节都应该有其存在的意义，并能为故事发展服务。
```

## 情节生成提示词模板

### 基础情节生成模板
```
你是一个经验丰富的故事架构师。请根据以下信息创建一个完整的故事大纲：

故事主题：{theme}
故事长度：{storyLength}
核心冲突：{coreConflict}
目标受众：{targetAudience}

请创建包含以下元素的故事大纲：

## 1. 故事设定
- 时间和地点
- 世界观和背景
- 社会环境和规则

## 2. 主要角色介绍
- 主角及其目标
- 反派及其动机
- 重要配角的作用

## 3. 三幕结构
### 第一幕：建立（约25%）
- 开场勾子
- 主角介绍和现状
- 催化事件
- 第一个转折点

### 第二幕：对抗（约50%）
- 主角的行动和阻碍
- 中点转折
- 危机升级
- 最低谷时刻
- 第二个转折点

### 第三幕：解决（约25%）
- 高潮对决
- 结局
- 新平衡

## 4. 关键转折点
至少3个重要的情节转折，包括：
- 事件描述
- 对角色的影响
- 推进故事的作用

## 5. 子情节线索
- 浪漫线索
- 角色成长线索
- 悬念线索

## 6. 主题深化方式
如何通过情节体现和深化主题

确保情节逻辑清晰，节奏把控得当，能够引起读者的情感共鸣。每个情节点都应该推进故事或揭示角色。
```

### 分支情节生成模板
```
作为一名交互式故事设计专家，请为以下情节节点创建多个分支选择：

## 当前情节状态
- 故事进度：{storyProgress}
- 主要角色状态：{characterStatus}
- 当前场景：{currentScene}
- 已发生的关键事件：{keyEvents}

## 分支节点设定
{branchDescription}

请为这个节点创建3-5个不同的发展方向，每个分支应包含：

### 分支选项格式：
**选项[数字]：[选择标题]**
- **短期后果**：立即发生的情况
- **角色影响**：对主要角色的影响
- **情节推进**：对主线剧情的作用
- **长期影响**：对后续故事的潜在影响
- **风险评估**：可能的危险或机会
- **情感权重**：引发的情感反应

确保每个分支都：
1. 符合角色性格和动机
2. 推进或深化故事
3. 保持情节张力
4. 为读者提供有意义的选择
5. 体现不同的价值观或策略

分支之间应该有明显差异，避免选择过于相似或无关紧要。
```

## 场景描述提示词模板

### 基础场景生成模板
```
你是一个擅长环境描写的作家。请根据以下信息创建一个生动的场景描述：

场景类型：{sceneType}
时间设定：{timeOfDay}
天气状况：{weather}
场景要素：{sceneElements}
情节功能：{plotFunction}
角色情绪：{characterMood}

请创建一个详细的场景描述，包含：

## 1. 感官描述
### 视觉（最重要）
- 整体布局和空间感
- 颜色搭配和光线效果
- 重要物件和细节
- 动态元素（移动的物体、变化的光影）

### 听觉
- 环境音效
- 特殊声音
- 音量层次

### 嗅觉和触觉
- 空气的味道和质感
- 温度和湿度
- 表面材质

## 2. 氛围营造
- 整体情绪基调
- 紧张感或舒适感
- 神秘感或开放感

## 3. 象征意义
- 场景如何反映角色内心
- 与主题的关联
- 预示性元素

## 4. 互动可能性
- 角色可能的行动
- 场景中的机会或威胁
- 推进情节的元素

描述要生动具体，让读者仿佛身临其境。避免过于抽象的表达，多用具体的细节。每个描述都应该为故事服务，而不仅仅是装饰。
```

## 对话生成提示词模板

### 基础对话生成模板
```
你是一个对话写作专家。请根据以下信息创建自然流畅的角色对话：

## 对话设定
- 对话类型：{dialogueType}
- 参与人数：{participantCount}
- 场景环境：{sceneContext}
- 对话目标：{dialogueGoal}
- 情节功能：{plotFunction}

## 角色信息
{characterDescriptions}

## 对话要求
请创建符合以下标准的对话：

### 1. 角色声音独特性
- 每个角色都有独特的说话方式
- 词汇选择反映背景和教育程度
- 语调体现性格特征
- 句式长度和结构的个人特色

### 2. 情节推进功能
- 透露重要信息
- 推进故事发展
- 建立或改变角色关系
- 制造或解决冲突

### 3. 情感层次
- 表层意思vs深层含义
- 言外之意和潜台词
- 情感的渐进变化
- 冲突的升级或缓解

### 4. 真实感
- 自然的对话节奏
- 合适的停顿和打断
- 日常化的语言特征
- 避免过于刻意的解释

### 5. 视觉化元素
- 适当的动作描述
- 表情和肢体语言
- 环境互动
- 道具使用

## 输出格式
```
[角色名]：[对话内容] *[动作/表情描述]*
[角色名]：[对话内容] *[动作/表情描述]*
...
```

对话要听起来真实可信，避免过于说教或刻意。每句话都应该有其存在的意义，推进情节或揭示角色。
```

### 冲突对话专用模板
```
作为一名冲突对话专家，请创建一段充满张力的对话：

## 冲突设定
- 冲突性质：{conflictType}（价值观、利益、误解、权力等）
- 冲突强度：{intensityLevel}（1-10级）
- 预期结果：{expectedOutcome}（加剧、缓解、转化等）

## 参与者分析
### 角色A：{characterA}
- 立场：{positionA}
- 情感状态：{emotionalStateA}
- 目标：{goalA}
- 弱点：{weaknessA}

### 角色B：{characterB}
- 立场：{positionB}
- 情感状态：{emotionalStateB}
- 目标：{goalB}
- 弱点：{weaknessB}

## 冲突对话结构

### 第一阶段：试探（2-3轮对话）
- 表面客套下的暗潮涌动
- 初步立场表达
- 寻找对方破绽

### 第二阶段：交锋（4-6轮对话）
- 直接冲突爆发
- 论点和反驳
- 情绪升温

### 第三阶段：激化（3-4轮对话）
- 触及核心分歧
- 个人攻击或深层伤害
- 情绪达到顶点

### 第四阶段：转折/结束（1-2轮对话）
- 意外变数或干预
- 新信息揭露
- 一方妥协或暂时退让

## 写作技巧要求
1. **节奏控制**：从慢到快，从隐到显
2. **情绪递进**：每轮对话都应该推高情绪强度
3. **真实反应**：角色的反应要符合其性格和背景
4. **留白艺术**：有些话不说完，让读者自己理解
5. **肢体语言**：用动作和表情强化对话效果

确保冲突合理可信，避免为了冲突而冲突。每个争论点都应该有其深层原因。
```

## 提示词优化指南

### 1. 提示词结构优化
- **角色设定**：明确AI的身份和专业背景
- **任务描述**：清晰说明要完成的具体任务
- **输入信息**：结构化提供必要的背景信息
- **输出要求**：详细说明期望的输出格式和内容
- **质量标准**：明确内容质量的评判标准

### 2. 语言风格优化
- 使用具体而非抽象的词汇
- 提供示例和反例
- 使用积极正面的表达
- 避免模糊和歧义的表述

### 3. 创意激发技巧
- 加入"意外元素"要求
- 鼓励"颠覆常规"思考
- 提供多个角度和可能性
- 要求解释创意来源和逻辑

### 4. 个性化定制
- 根据用户输入动态调整
- 提供多个难度级别选项
- 适应不同故事类型和风格
- 支持文化背景定制

### 5. 迭代改进机制
- 收集用户反馈
- A/B测试不同提示词版本
- 分析高质量输出的共同特征
- 持续更新和优化模板
